# Screenshots — Verilink Solutions CRM Project

This folder contains all screenshots taken during the 
Verilink Solutions CRM Analytics, Customer Success 
Intelligence and AI Automation project.

Screenshots are organized into subfolders by tool.

---

## Folder Structure

| Subfolder | Contents |
|-----------|---------|
| HubSpot | CRM setup, contact management, customer support activities, tickets, deals, dashboard, email templates, snippets, pipeline, segments |
| LibreOffice_Calc | Data cleaning, summary tab, churn analysis, dashboard charts, observations |
| MySQL_Workbench | Database setup, CSV import, all 6 SQL query results |
| n8n_Automation | Workflow canvas, node configurations, automation test results |
| GitHub | Repository structure at key milestones |

---

## Naming Convention

All screenshots follow this format:

`Verilink_[Tool]_[WhatItShows].png`

Examples:
- `Verilink_HubSpot_pipeline_with_deals.png`
- `Verilink_LibreCalc_churn_analysis_complete.png`
- `Verilink_MySQL_query3_tenure_churn.png`
- `Verilink_n8n_workflow_complete.png`
